運用支援サービスとは

ITインフラの運用を経験豊富な技術者がご支援します。
定常的な運用タスクだけではなく、業務効率化のための自動化や各種ツールも組み合わせてお客様に最適＆期待以上の付加価値を提供できるよう日々取り組んでおります。

サービス対象範囲

以下の環境・ソリューションを中心に運用業務をご支援しておりますが、柔軟な対応も可能です。


PC運用・ユーザーサポート

エンドユーザー様が使用されるPCや業務システムのサポートや、OA・システム部門で行われるPC周りの各種運用業務を行います。


＜技術支援＞

  • Windows PCに関わる各種対応（セットアップ / キッテング / インストール / 設定 / 設置）
  • Windows PC マスター作成及び展開（Ghost / WDS)
  • 業務アプリケーションに関わる各種対応（セットアップ / 設定 / 障害切り分け）
    • ※対応可能な業務アプリケーション例（SAP / Notes Domino / ES/1 / Lanscope）

＜運用支援＞

  • コールセンター / ヘルプデスク / オンサイト
  • ベンダー修理依頼等

サーバ・システム運用支援

主にWindows系システムやVMware環境の監視や管理を行います。


＜監視＞

  • Microsoft製品によるシステム基盤（Windows Server / Hyper-V）
  • VMware製品によるシステム基盤（vSphere Server / vSphere ESX / vSphere ESXi）

＜管理＞

  • Microsoft Windows Server の各種機能（Active Directory / SCCM / WSUS / DFS）
  • 運用ドキュメント（操作手順書 / 復旧手順書 / 対応フロー / 申請書等）
  • バックアップ、リカバリ（Windows Backup / Symantec Backup Exec / CA Arcserve）

資産管理

＜監視＞

  • ハードウェア管理（起票 / 更新 / 調達 / 破棄)
  • ソフトウェア管理（起票 / 更新 / 調達 / 破棄）

その他

＜ベンダー支援＞

  • 製品保守サポート（ユーザーサポート / 導入支援等）

＜運用コンサル＞

  • 運用改革支援（現状分析 / 課題の洗い出し / 改善提案 / 施策実施等）
  • システムパフォーマンス監視支援（ES/1等を使用したパフォーマンス監視 / 報告書作成等）

サービス形態

サービス形態 特徴 留意事項
お客様先常駐 貴社の事務所に技術者が常駐し、座席やPC、運用ツール等を提供頂き、所定の運用業務を実施
定期訪問 常駐するほどの業務ボリュームは無いが、定期的に発生する運用タスクや定期点検作業を貴社社員に代わって必要なボリュームに応じて運用を実施
遠隔システム保守 弊社がインフラ構築したシステムを対象に電話とメールによるシステム保守対応 オンサイト対応は別途費用が生じます

サービス対象地域

1都3県（東京都、埼玉県、神奈川県、千葉県）を中心にサービスを提供
但し、条件により日本全国及びASEAN地域でのサービスも可能です、弊社営業までご相談ください

契約形態

以下のいずれも対応可能です


契約形態 特徴 適している業務/環境
派遣契約
  • 最小1名から対応可能
  • 勤務地は貴社事務所内
  • 技術者への指揮命令は貴社
  • 精算形態は稼働実績時間により変動
  • 成果物完成責任は無し
  • 都度個別の対応で指示を要する対応
請負契約
  • 最小2名から対応可能
  • 勤務地は貴社事務所内/弊社事務所のいずれも可能
  • 技術者への指揮命令は弊社責任者
  • 精算形態は成果物に対しての固定額
  • 成果物完成責任は有り
  • 業務内容が決まっており変動がないケース
  • 業務の処理方法に決まりが無く、ベンダーに任せられるケース
準委任契約
  • 最小1名から対応可能
  • 勤務地は貴社事務所内/弊社事務所のいずれも可能
  • 技術者への指揮命令は弊社責任者
  • 精算形態は稼働実績時間により変動
  • 成果物完成責任は無し
  • 上流工程の要件定義等で事前に明確な範囲（工数）を算出できないが、自社にノウハウが無く指揮命令もできないケース


解決できる課題

  • 運用業務の内容、スキルレベル、負荷状況を把握したい
  • 事業の海外展開に伴い、現地のサポート体制を構築する必要がある
  • 中・短期のプロジェクトに伴う一時的な負荷増に柔軟にリソースを配置したい
  • 日々の運用タスクをこなすだけではなく、課題の発見～解決を定常的なプロセスで実施したい
