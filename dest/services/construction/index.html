インフラ構築サービス

MicrosoftとVMwareソリューションによる
インフラ構築はIHSにお任せください！

サービス概要

お客様のご要望等に合わせ、インフラ基盤の構築に関する要件定義からドキュメント作成までをサポートいたします。特にWindowsを始めとするMicrosoftソリューションと仮想化ソリューションのVMwareで豊富な実績がございます。クラウドサービスではMicrosoft Azureに強みがございます。

対応可能な製品・機能一覧

主なソリューション別構築サービス

エンタープライズITの中でも重要度が高く、多くのご相談を頂いているソリューションに注力し、インフラ基盤の設計構築をご支援いたします。
特にIHSで実績が多い構築事例は以下の通りです。


Active Directory・Exchange Server構築サービス

認証機能を提供するActive Directory及びメッセージプラットフォームであるExchange Serverの設計、構築をサポートいたします。
新規導入はもちろん、拡張、バージョンアップ、他社ソリューションからの移行まで幅広い対応が可能です。近年ますます増大するセキュリティ脅威の中、セキュアな認証基盤を実現するActive Directory環境の構築に期待が高まっています。

Active Directory・Exchange Server構築サービスの導入事例

System Center Configuration Manager（SCCM）構築サービス

Microsoftの提供するSystem Center製品群の一つであるSCCMの構築をサポートいたします。
IT資産情報の管理、セキュリティパッチ配信やSCEP(Endpoint Protection)の管理機能等、SCCMの導入により、セキュアなクライアントコンピュータの管理が可能となります。
また、SCCMはWindows10から採用されたCBBによる機能アップデートでも重要な役割を果たします。

System Center Configuration Manager（SCCM）構築サービスの導入事例

仮想環境構築サービス

近年、注目されている仮想化技術は、ハードウェアへの投資コストの削減、レガシーOSとの共存、可用性の向上等、様々なメリットがあり、システム部門のみならず経営者にも注目されています。
お客様の用途に応じたVMwareによる仮想環境の構築を、規模の大小を問わずサポートいたします。

仮想環境構築サービスの導入事例

Horizon VIEWによるVDI環境構築サービス

VMwareが提供するVDIソリューションであるHorizon VIEWによる仮想デスクトップ環境の構築は、拠点ごとの仕組み・体制の統一、クライアントの一元管理等、管理の簡素化とコストの削減が可能となり、ROIを大幅に向上できます。
VMwareの構築で培ったノウハウを利用し、仮想デスクトップ環境の設計、構築をサポートいたします。

Horizon VIEWによるVDI環境構築サービスの導入事例

IHSが選ばれている理由

  • MicrosoftとVMwareソリューションに多数の実績

    100人以下の会社の単独ソリューションでのシステム導入から100台規模のESXサーバによる仮想環境の導入、数千ユーザーの仮想デスクトップ環境の構築、数万台のクライアントを管理するADやSCCMの構築など、幅広い規模で豊富な実績がございます。

  • 新入社員教育から一貫した教育体制のもと、多くの有資格者が在籍

    IHSの社員は、入社後の新人研修でMicrosoftの認定技術資格であるMCPを最新のサーバOSを対象に取得しており、VMware認定資格であるVCPの有資格者も複数名在籍しております。
    また、豊富な知識と経験を有するエンジニアが全案件においてサポートできる体制をとっています。

    • MCP取得人数：83名
    • VCP取得人数：5名

  • 柔軟な対応

    • 契約形態：一括請負でも工数清算の準委任契約でも可能
    • 場所：日本全国はもちろんの事、海外でのインフラ導入も可能（全て東京からの出張扱い）
    • 工程：要件定義～運用引き渡しまで全体から、一部工程のみの支援も可能
