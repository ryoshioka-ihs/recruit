WebシステムAzure移行サービスとは

クラウドと相性の良いWebシステムのMicrosoft Azureへのシステム移行をご支援します。海外へのサービス提供や負荷に応じた柔軟なスケーリングはクラウドを活用してこそ迅速安価に実現します。


解決できる課題

販促キャンペーンサイトを立ち上げたい時にシステム立ち上げに時間が掛かり、タイミングを逃してしまった
サーバの増強は5分も掛からずに完了！ロードバランシングも複雑な設定は不要！
システムリソースを高負荷時にあわせて調達しており、通常時のリソース稼働率が低い
稼働状況に合わせたリソースの最適配置もクラウドだから容易！
コストもニーズにあわせて最適化！
リリース後にパッチ適用等の対策を十分に行えていない
検証環境の迅速な立ち上げで素早くテストを行い、本番適用までの時間を短縮！
コストの制約で開発環境を持てない
開発環境は必要な時だけクラウド上で起動することでコストを最小化！
DRシステムも必要だが、費用対効果の観点で投資に踏み切れない
Azureのサービスを利用してコールドスタンバイのDR環境を安価に用意！

Azureを選択する理由

  • グローバル19か所からビジネス・システム要件に最適なセンターを選択可能

    日本国内では関東・関西の2か所を利用可能

  • Microsoftソリューションでオンプレミスとシームレスな統合運用を実現

    ハイブリッドクラウド環境の運用をSystem Center製品で効率化

  • IaaSだけでなくAzureのPaaSを組み合わせることで高い付加価値を提供

    認証のAzure ADやバックアップ、DR、機械学習等の多様なサービス

  • セキュアな基盤

    アメリカで国防総省に次いで攻撃を受けるMicrosoftだからこその対策と実績

  • 自動化の仕組みで効率的な運用を実現

    Azure AutomationやPowerShellワークフロー（Runbook）で運用タスクを自動化

サービスメニュー

標準サービス

  • 現行システムの環境、構成把握
  • Azure上でのシステム構成の検討（IaaS利用/PaaS利用等）
  • Azureの契約代行
  • Azure上での環境構築
  • Azureへのデータ移行
  • システム切替

オプションサービス

System Centerを活用したオンプレミスとAzureの統合運用

＜利用製品＞

  • System Center Operation Manager：システム監視
  • System Center Virtual Machine Manager：仮想環境管理
  • System Center App Controller：Azureとオンプレの統合管理

自動化機能を利用した運用タスクの自動化

＜利用製品＞

  • Azure Automation

    • …Windows PowerShellワークフローによってAzureリソースおよびサードパーティ製アプリケーションの作成、デプロイ、監視、保守を処理

  • PowerShell Desired State Configuration

    • …クラウドやオンプレミスにある物理マシンや仮想マシンといったリソースのあるべき状態の構成を、容易でありながら信頼性の高い方法で監視し、自動的に更新
