よくある質問

特徴について

どのような業務に向いていますか？
あるいは活用できますか？

基本はWindows上のすべての作業が対象になります。
ウェブやExcel、Wordといったアプリケーションの他、基幹システムへのデータ入力、複数システム間でデータ連携など、各種実績があります。
利用されている業種も、金融、製造、運輸、ソフト開発など様々です。

主な特徴を教えてください。

クライアント型のアプリケーションなので、専用サーバーがなくてもクライアントPCにインストールするだけですぐに使用できます。
また国産のアプリケーションなので、画面やマニュアルが理解しやすく、開発元のサポートも国内で完結します。

誰でも容易にシナリオを作ることができますか？

プログラミング経験があれば容易に理解できシナリオの作成ができると思われます。
また専用のチュートリアルを試し、操作マニュアルを利用すれば短期間でできるようになると思われます。
さらに弊社のサポートサービスを利用していただければ、より確実に作成できます。

導入するとどの程度の改善効果が見込めますか？

業務や対象データなど状況によりますが、30%程度の改善率が多く、90%以上の例もあります。

対象アプリについて

IE以外のブラウザも対応可能ですか？

FirefoxやChromeなどは画像認識で操作することができますが、IEには専用のノードが多数用意されており、より容易に開発ができます。

社内で使用しているシステムがWinActorでウィンドウ認識できません。
どういう理由でしょうか？

ターゲット選択しても何も反応しない、選択時のオレンジ枠が出ても反応しないなどの場合は、対象システムに保護モードがかかっている、上位権限になっていることがあります
保護モードを解除するか、WinActorを管理者として実行で解消できる可能性があります。

機能について

作成・保存できるシナリオの数に制限はありますか？

作成・保存できるシナリオの数に制限はございません。

マウスの操作は記録できますか？

エミュレーションモードという機能で記録できます。

ウェブ画面上のデータ(数値や文字)はどうやって取得できますか？

Internet Explorer(IE)では複数の方法が用意されています。(表の値取得、文字列取得、HTMLソースから取得など)
他に画像マッチングで人が操作するように該当データクリックして取得する方法もあります。

ウェブ画面上のデータ(数値や文字)はどうやって取得できますか？

Internet Explorer(IE)では複数の方法が用意されています。(表の値取得、文字列取得、HTMLソースから取得など)
他に画像マッチングで人が操作するように該当データクリックして取得する方法もあります。

実行時にエラーが発生したらどう対処すればよいですか？

ログインやパスワードの入力では、画像マッチングで該当画面を表示させて入力フィールドクリックし、クリップボードからIDを貼り付ける、というような操作で容易に作ることができます。

ローカルPC内のWinActorでリモート接続先のPCを操作できますか？

可能です。
画像マッチングやエミュレーションモードの操作はできますが、IEモードやイベントモードは使用できません。
ただし仮想環境では容量の問題、操作の制限など制約がかかっていることがあるため、事前に試用版を使って十分確認をしてください。

高解像度のディスプレイでWinActorを起動すると文字がはみ出したり、アイコンがはみ出したりする表示崩れが発生します。対策を教えてください。

現在ディスプレイのテキストサイズの変更に追随できない仕様となっております。
ディスプレイのカスタマイズ画面でサイズを100%に変更してから、WinActorを再立上げしてください。

並列操作は可能でしょうか？

基本的にWinActorは1台のPCに1プロセスしか起動できません。シナリオについても並列実行はできません。。
WinActorが自動操作を行っている最中は、WinActorがPCを占有するとお考え下さい。そのため、WinActorによる自動操作中に人がそのPCを操作してしまうと誤操作、誤投入の原因となります。
並列実行を行うためには、PCを複数台ご準備いただくことになります。

環境について

仮想環境で動かすことはできますか？

可能です。
ただし仮想環境では容量の問題、操作の制限など制約がかかっていることがあるため、事前に試用版を使って十分確認をしてください。

サーバー上にインストールして利用できますか？

Windows Server 2016に対応しており、利用可能です。
ただしライセンスは使用される方のプロファイル(ユーザーデスクトップ)ごとに必要となります。

MacやLinuxでも稼働しますか？

どちらの環境も稼働しません。
現在稼働するOSはWindows7、8.1、10、Windows Server 2016となっています。

スクリーンロック状態で動作しますか？

スクリーンロック状態では動作しません。
スクリーンロック、スクリーンセーバは、オフにした状態で運用してください。

管理について

シナリオを定期的、または時間指定で実行することはできますか？

Windows標準のスケジューラに設定して実行することができます。
また、WinActorを常に動作させ、機能の指定時間待機で時間(09:00開始、09:00-18:00の間だけ動作等)を指定し、繰り返し実行することで対応可能です。
WinActorを常に動作させる場合は、他のシナリオは実行できませんのでご注意ください。

試用版を使ってシナリオを作成しましたが期限切れになりました。製品版でも使えないのでしょうか？

試用版ではシナリオ情報に有効期限が自動的に設定されます。
期限切れになったシナリオは、製品版を導入すればその有効期限を解除することができ、引き続き使用することができます。

バージョンアップについて

バージョンアップの費用を教えてください。

マイナーバージョンアップは保守費用内で対応できますが、メジャーバージョンアップでは別途費用がかかります。

試用版を使ってシナリオを作成しましたが期限切れになりました。製品版でも使えないのでしょうか？

使用できます。
マイナー、メジャーバージョンアップでもシナリオは上位互換になっています。

ライセンスについて

ライセンスに種類はありますか？

シナリオの作成および実行が可能な「フル機能版」と、シナリオ実行専用の「実行版」がございます。

利用しているPCが壊れたり、組織変更などでPCの変更があるときはライセンスをどう移行できますか？

条件によりライセンスの再発行ができます。該当の手続きが用意されていますので申請してください。

サポートについて

シナリオの開発を依頼することはできますか？

弊社のサポートで引き受けることが可能です。
